03-accesskey
The accesskey attribute specifies a shortcut key to activate/focus an element.
The accesskey attribute value must be a single character (a letter or a digit).

Note:
To avoid the problem of conflicting with other key browser patterns, most browsers 
will use access keys only if pressed along with the Alt key.

It is not advisable not to use access keys because adapting them to all international 
languages is difficult because the value of the access key may not be present on all keyboards

Syntax
<element accesskey="character">

Example:
<a href="https://www.w3schools.com/html/" accesskey="h">HTML</a><br>
<a href="https://www.w3schools.com/css/" accesskey="c">CSS</a>
